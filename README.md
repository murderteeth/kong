# Kong
Real-time and historical ZooTroop GraphQL API

```bash
yarn
cp .env.example .env
# configure .env
make build
make dev
```

## tmux cheats
`quit` - `ctrl+b`, `:` then `kill-session` (your dev environment will also shutdown gracefully)
`pane navigation` - `ctrl+b` then `arrow keys`
`zoom\unzoom pane` - `ctrl+b` then `z`
